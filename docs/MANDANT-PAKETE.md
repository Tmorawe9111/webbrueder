# Mandanten-Pakete (intern) — Webbrüder

Dieses Dokument ist **nicht** für die öffentliche Website. Es dient als Vorlage für Angebote und Kickoffs — abgestimmt auf die Leistungen auf webbrüder.de.

---

## 1. Marketingstrategie

**Ziel:** Klarheit über Zielgruppe, Positionierung und Kanäle, bevor investiert wird.

| Phase | Inhalt | Typische Dauer |
|-------|--------|----------------|
| Kickoff | Ziele, Ist-Zustand, Wettbewerber | 1 Termin |
| Analyse | Website, Social, Marktumfeld | 3–5 Tage |
| Strategie | Positionierung, Messaging, Kanäle | 3–5 Tage |
| Roadmap | 90-Tage-Content-Plan | 2–3 Tage |

**Abgrenzung:** Keine laufende Posting-Pflege (→ Social Media). Keine technische Website-Umsetzung (→ Website / Neustart).

**Kickoff-Fragen:**
- Was soll in 6–12 Monaten online erreicht werden?
- Wer ist die Kernzielgruppe?
- Welche Kanäle nutzen Sie heute — und welche Wirkung sehen Sie?
- Wer sind 2–3 Wettbewerber oder Vorbilder?

---

## 2. Social-Media-Betreuung

**Ziel:** Regelmäßiger, professioneller Auftritt auf den passenden Kanälen.

| Leistung | Option A (Strategie) | Option B (Betreuung) |
|----------|----------------------|----------------------|
| Kanalwahl & Strategie | ✓ | ✓ |
| Redaktionsplan | ✓ | ✓ |
| Templates / Visuals | ✓ | ✓ |
| Posting & Veröffentlichung | — | ✓ |
| Community (Kommentare/DMs) | — | im Rahmen |
| Monatsreport | — | ✓ |

**Abgrenzung:** Keine bezahlten Ads (Media Budget separat). Rechtliche Prüfung Heilmittelwerbung etc. beim Mandanten bzw. Fachanwalt.

**Kickoff-Fragen:**
- Welche Profile existieren bereits?
- Haben Sie Bild-/Video-Material vor Ort?
- Wie viel Zeit können Sie für Freigaben aufbringen?
- Gibt es No-Gos (Themen, Tonalität)?

---

## 3. Digitaler Neustart (Flaggschiff)

**Ziel:** Einheitlicher Markenauftritt — Strategie → Konzeption → Design → Umsetzung → optional Betreuung.

**Phasenmodell (Vertrag):**

1. **Strategie** — Marketingstrategie (siehe oben)
2. **Konzeption** — Struktur Website + Social, Content-Plan
3. **Design** — Visuelles Konzept, einheitliche Markenführung online
4. **Umsetzung** — Website, Social-Setup, SEO/GEO-Grundlage
5. **Livegang** — Abnahme, Einweisung
6. **Betreuung** (optional, monatlich kündbar) — Social und/oder Website-Updates

**Typischer Zeitrahmen:** 6–12 Wochen (Gesamtprojekt).

**Kickoff-Checkliste (Auszug):**
- [ ] Unternehmensbeschreibung in 3 Sätzen
- [ ] Zielgruppe & Region
- [ ] Bestehende Website / Social-URLs
- [ ] Logo, Farben, bestehende CI-Materialien
- [ ] Wettbewerber / Inspiration
- [ ] Ansprechpartner & Freigabeprozess
- [ ] Gewünschter Live-Termin (falls fest)
