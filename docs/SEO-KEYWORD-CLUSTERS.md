# Keyword-Cluster (intern) — Webbrüder

Eine URL = eine Hauptintention. Die **Startseite** [`index.html`](../index.html) trägt das breite Leistungs- und Markencluster; **Unterseiten** schärfen Einzelthemen.

| Haupt-URL | Hauptintention (1 Satz) | Unterstützende Themen / Long-Tail |
|-----------|-------------------------|-------------------------------------|
| `https://webbrüder.de/` | Webseiten erstellen lassen mit Webdesign, SEO und GEO für KMU in Deutschland | Webdesign Frankfurt, Webentwicklung, ohne Baukasten, DSGVO |
| `https://webbrüder.de/gastronomie-webseite.html` | Website / Webseite für Gastronomie (Restaurant, Speisekarte, Reservierung) | Webseite Restaurant, Gastro Website, lokal SEO |
| `https://webbrüder.de/impressum.html` | Rechtliche Anbieterkennung | — |
| `https://webbrüder.de/datenschutz.html` | Datenschutzerklärung | — |

## Cluster-Beispiele (Content-Richtung)

1. **Transactional / Dienstleistung:** „Webseite erstellen lassen“, „Website erstellen lassen“, „Webdesign Agentur“ (Frankfurt als regionaler Zusatz, nicht als einziges Keyword).
2. **Branche Gastronomie:** „Restaurant Website“, „Webseite Gastronomie“, „Speisekarte online“, „Reservierung Website Restaurant“.
3. **Branche Handwerk:** „Handwerk Website“, „SHK Webseite“, „lokales Handwerk online“.
4. **SEO/GEO:** „technische SEO“, „GEO KI-Suche“, „Suchmaschinenoptimierung Website“ — kurz in Leistungstexten, ohne Keyword-Stuffing.

## OnPage-Hinweis

H2 und erste Absätze je Sektion sollten die **Intention der Sektion** in einem Satz tragen (Portfolio = Referenzen, Branchen = Zielgruppen, Leistungen = Lieferumfang, usw.). Änderungen an Überschriften immer mit Lesbarkeit und Markenstimme abgleichen.
